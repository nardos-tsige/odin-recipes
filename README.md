# Odin Recipes

A simple HTML recipe website built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## Live Demo

[View Live Demo](#) (https://github.com/nardos-tsige/odin-recipes)

## Features

- Home page with links to multiple recipes
- Individual recipe pages with:
  - High-quality food images
  - Recipe descriptions
  - Ingredients lists
  - Step-by-step cooking instructions
  - Navigation links back to home
